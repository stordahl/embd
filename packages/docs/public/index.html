Embd

a framework for 3rd Party UI

This button is embedded 3rd Party JS built with Svelte.js & esbuild

Simply create a target element, import the JS bundle into your page, and create a new instance of the widget...


<div id="target-div" />
<script src='dist/bundle.js'></script>
<script>
  const widget = new Embd.Widget({
    target: "target-div",
    props: {
      text: "Click Me",
      callback: () => alert('Passing a callback!')
    }
  })
</script>
